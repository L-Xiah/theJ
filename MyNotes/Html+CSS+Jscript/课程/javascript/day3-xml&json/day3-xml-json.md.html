XML

什么是XML

  • XML 指可扩展标记语言（EXtensible Markup Language）
  • XML 是一种标记语言，很类似 HTML
  • XML 的设计宗旨是传输数据，而非显示数据
  • XML 标签没有被预定义。您需要自行定义标签。
  • XML 被设计为具有自我描述性。
  • XML 是 W3C 的推荐标准

XML和HTML的区别

  • XML和HTML并没有直接关系（不存在谁替代谁）
  • XML主要是用来做数据的存储和传输，而HTML主要是用来显示
  • XML就是纯粹的文本文件，只不过是符合了标准格式，完全可以自定义
  • XML的标签必须是闭合的
<?xml version="1.0" encoding="ISO-8859-1"?>
<root>
  <node0 attr1="abc" attr2="bcd">
    <node1></node1>
  </node0>
</root>

一个标准的XML应当包含如下内容：

  • 一个XML声明
  • 有且仅有一个根节点
  • 可自定义但必须闭合的标签
  • 可自定义的属性

NOTE:

  1. XML对大小写也是敏感的，如<node></noDE>是不允许的
  2. 标签必须按反向顺序闭合
  3. 由于XML中包含特殊符号如大于小于号，所以需要使用转义符来代替
  4. 在XML中，空格会被完整保留

XML注释

XML中注释与HTML相同，也是<!-- -->

XML的元素

XML 元素指的是从（且包括）开始标签直到（且包括）结束标签的部分。元素本身可以包含其他元素、文本或者混合物，同时也可以拥有属性。

  • 元素包含文本
  • 元素属性
  • 元素嵌套元素

XML的命名规则：

  • 名称可以含字母、数字以及其他的字符
  • 名称不能以数字或者标点符号开始
  • 名称不能以字符 “xml”（或者 XML、Xml）开始
  • 名称不能包含空格
  • 单引号和双引号都是被允许的，嵌套的时候使用单双引号交替，或转义符
  • 元素本身为空时，可以用/>关闭而不用写后面的标签 NOTE：尽量不要以-, ., :进行标签的命名

  • getElementsByTagName("tag") 加载标签
  • childNodes 加载孩子标签
  • nodeValue 加载标签的值 / textContent 获取文本内容 / innerHTML
  • getAttribute("attr") 获取具体属性值
  • length 获取长度

修改xml

  • createElement() 创建元素节点
  • createTextNode() 创建新的文本节点
  • parentNode 获取父节点
  • appendChild() 向节点添加子节点
  • insertBefore() 向指定节点之前添加子节点
  • removeChild() 删除指定的节点
  • setAttribute() 设置属性值
  • removeAttribute() 删除属性
  • replaceChild() 替换节点

JSON

JSON(JavaScript Object Notation) 是一种轻量级的数据交换格式。它基于ECMAScript的一个子集。 JSON采用完全独立于语言的文本格式，但是也使用了类似于C语言家族的习惯（包括C、C++、C#、Java、JavaScript、Perl、Python等）。这些特性使JSON成为理想的数据交换语言。 易于人阅读和编写，同时也易于机器解析和生成(一般用于提升网络传输速率)。

{
  "employees": [
    { "firstName":"Bill" , "lastName":"Gates" },
    { "firstName":"George" , "lastName":"Bush" },
    { "firstName":"Thomas" , "lastName":"Carter" }
  ]
}

对于json的构成，可以包括大部分语言中的基本类型如数字int、double、 float； 逻辑值true、false；字符串类型；数组[]；对象{}。以键值对的形式，用,逗号隔开。比如下面的实例：

// 一般对象：
{
  "sid": 1,
  "sname": "James",
  "birth": "1995-08-08",
  "address":"Jiaxing"
}
// 数组对象：
[
  {"sid": 1, "sname": "James"},
  {"sid": 2, "sname": "Iverson"},
  {"sid": 3, "sname": "Garnett"}
]
// 嵌套对象：
{
  "school_name": "Infosys",
  "student": [
    {
      "sid": 1,
      "sname": "James",
      "score": {
        "Java": 86.5,
        "Oracle": 90,
        "HCJX": 35
      }
    },
    {
      "sid": 2,
      "sname": "Tom",
      "score": {
        "Java": 75.5,
        "Oracle": 68,
        "HCJX": 90
      }
    }
  ]
}
  • 对象{},数组[]
  • 字符串键值对"key":"value"，数字键值对"key": value
  • 以','逗号来分割各个K-V项，最后一个不加

JS中解析JSON的方法：

  • eval() "(" + target + ")"
  • JSON.parse()
var jsonstr = "{'school_name':'Infosys','student':[{'sid':1,'sname':'James','score':{'Java':86.5,'Oracle':90,'HCJX':35}},{'sid':2,'sname':'Tom','score':{'Java':75.5,'Oracle':68,'HCJX':90}}]}";
 
// eval 解析
var jsonobj = eval( "(" + jsonstr + ")" );
// 输出第一个学生的oracle成绩
alert("学生[" + jsonobj.student[0].sname + "]的Oracle成绩是[" + jsonobj.student[0].score.Oracle + "]分");
 
// JSON解析
var jsonobj = JSON.parse(jsonstr);
// 输出第一个学生的oracle成绩
alert("学生[" + jsonobj.student[0].sname + "]的Oracle成绩是[" + jsonobj.student[0].score.Oracle + "]分");
