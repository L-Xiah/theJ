Ajax

什么是Ajax

AjaxAsynchronous JavaScript and XML，是指一种创建交互式网页应用的网页开发技术。

  • 通过javascript来实现和后台的交互，传输XML格式的数据
  • 无需刷新页面，通过操作DOM来实现重新加载部分页面上的数据 NOTE: 同步请求vs异步请求

XMLHttpRequest

XMLHttpRequest是Ajax技术的核心，首先它是建立在HTTP协议的基础上，在IE5中首次提出，通过使用XMLHttpRequest，可以使JavaScript直接向“后台”发起请求，而不阻塞“用户前台”。

var request = new XMLHttpRequest();
<!-- 真爱生命，远离IE6 -->
var request;
if(window.XMLHttpRequest) {
  request = new XMLHttpRequest(); // IE7+及其他浏览器
} else {
  request = new ActiveXObject("Microsoft.XMLHTTP"); // IE6, IE5
}

HTTP 请求中包含的信息：

  • 请求的方法和动作：GET, POST以及其他的类似PUT,DELETE等
  • 请求地址的URL路径
  • 请求头(header)，包含一些客户端环境信息，身份验证信息等
  • 请求体，即请求正文，包含自定义的查询信息，表单等等

HTTP 响应中包含的信息：

  • 一个数字和文字组成的状态码(404, 200, 503等等)
  • 响应头，与请求相对应，包含一些属性信息，如服务器类型，日期时间，内容类型和长度等
  • 响应体，即响应正文，返回值 状态码介绍： 1xx 信息类，表示web服务器收到了请求，正在处理中 2xx 成功类，表示正确接收和理解了请求 3xx 重定向，表示请未执行成功，客户端需要采取进一步操作 4xx 客户端错误，表示客户端提交的请求有误 5xx 服务端错误，表示服务器端无法处理请求
request.open("POST", "register.jsp", true);
request.setRequestHeader("Content-type", "application/x-www-form-urlencoded"); // 声明发送form表单
request.send("name=laoliu&sex=男");

通过readyState来监听服务器处理请求的变化:

  • 返回值为0：请求未初始化，open还没有调用
  • 返回值为1：服务器连接已经建立，open调用
  • 返回值为2：请求已经接受，获得响应头
  • 返回值为3：请求处理中，获得响应主体
  • 返回值为4：请求完成，响应就绪完成了
    var request = new XMLHttpRequest();
    request.open("GET", "login.jsp", true);
    request.send();
    // 监听状态
    request.onreadystatechange = function() {
    if(request.readyState == 4 && request.status == 200) {
      // 使用request.responseText做一些事情
    }
    }
